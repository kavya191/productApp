# React Web Application

Technology used:
HTML,CSS,Javascript,React

Dev Tool: Vite
Libraries : React-bootstrap  , Axios ,React-icon, React-feather
State Management : Context Api

Run Command :
  git clone
  npm install
  npm run dev
  h+enter
  o+enter
